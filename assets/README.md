# Assets

Coloca aquí los siguientes archivos:

- `icon.png` — 1024×1024 · ícono principal de la app
- `adaptive-icon.png` — 1024×1024 · ícono adaptativo de Android
- `splash.png` — 1284×2778 · imagen del splash nativo de Expo
- `notification-icon.png` — 96×96 · ícono monocromo (blanco sobre transparente) para notificaciones en Android
- `favicon.png` — 48×48 · favicon web

Durante desarrollo, Expo puede arrancar sin ellos, pero se recomiendan antes del build de producción.
